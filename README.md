## Descrição

[Nest](https://github.com/nestjs/nest) repositório inicial TypeScript da estrutura.

## Instalação

```bash
# Clonar o repositório
$ git clone https://github.com/SilvaneiMartins/nest-expense-app

# Acessar o diretório
$ cd nest-expense-app

# Instalar as dependências
$ yarn install
```

## Executando o aplicativo

```bash
# Desenvolvimento
$ yarn run start

# Modo de observação
$ yarn run start:dev

# Modo de produção
$ yarn run start:prod
```

## Testando o aplicativo

```bash
# Testes unitários
$ yarn run test

# Testes e2e
$ yarn run test:e2e

# Testes de cobertura
$ yarn run test:cov
```
